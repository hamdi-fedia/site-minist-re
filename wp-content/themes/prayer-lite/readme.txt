== Description ==
Prayer Lite is an impressive charity WordPress theme with a clean, modern and creative design.
 It is an attractive and fully responsive theme specially designed for churches, charity organizations and other kinds of religious websites. This multi-functional and highly adaptive theme can also be used for non-profit organizations, political organizations, foundations, social causes, ministry, donation, fundraiser, event and other related organization.
